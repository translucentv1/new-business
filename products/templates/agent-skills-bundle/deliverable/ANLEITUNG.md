# Agent-Skills Bundle – Anleitung

Danke für den Kauf. Du hast 7 fertige `SKILL.md`-Dateien erhalten – jede liefert
**immer das gleiche Ausgabe-Format**, damit du den Text direkt weiterverarbeiten
kannst (Copy-Paste, keine Einrichtung).

## So nutzt du die Skills

**Claude Code / Hermes (Skills-Ordner):**
Kopiere die gewünschten `.md`-Dateien in deinen `skills/`-Ordner (oder das
Skills-Verzeichnis deines Agents). Der Agent lädt sie bei Bedarf.

**ChatGPT (Custom Instructions / Projekt):**
Öffne eine `.md`-Datei unten, kopiere den gesamten Inhalt in deine
Projekt-Anweisungen oder einen neuen "Custom GPT" als Instruktion.

**Einmalig ausprobieren:**
Markiere den kompletten Inhalt einer `.md`-Datei und gib ihn als Prompt ein –
die Antwort folgt exakt dem im Skill definierten Format.

## Die 7 Skills

1. `commit-message-writer.md` – konventionelle Git-Commits
2. `ci-pipeline-trier.md` – Root-Failure eines kaputten Builds
3. `root-cause-debugger.md` – Symptom → Root Cause → Fix
4. `daily-standup-writer.md` – Standup-Notes aus Commits
5. `messy-data-cleaner.md` – Dedupe / Fix / Normalize von CSV
6. `test-case-generator.md` – Edge Cases & Coverage
7. `clean-code-reviewer.md` – Bugs vor dem Ship

Alle Skills sind modellagnostisch (Claude & ChatGPT). Kein Abo, kein Signup –
einmal kaufen, dauerhaft nutzen.
