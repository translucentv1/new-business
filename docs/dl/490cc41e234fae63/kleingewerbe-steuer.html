# Kleingewerbe Steuer-Planner (§19 UStG)

Zielgruppe: DE-Kleingewerbler mit Kleinunternehmerregelung

So nutzt du die Tabelle:
1. Oeffne budget.csv in Google Sheets oder Excel.
2. Trage deine Betraege in Spalte B ein.
3. Die Zelle 'SUMME' rechnet automatisch ( =SUMME(B2:B9) ).
