# Finanz-Tracker DACH (Monatsbudget)

Zielgruppe: DE-Sparer, die Ausgaben pro Monat im Blick behalten wollen

So nutzt du die Tabelle:
1. Oeffne budget.csv in Google Sheets oder Excel.
2. Trage deine Betraege in Spalte B ein.
3. Die Zelle 'SUMME' rechnet automatisch ( =SUMME(B2:B10) ).

Sektionen: Einnahmen (Gehalt, Nebenjobs, sonstiges) | Fixkosten (Miete, Versicherungen, Abos) | Variable Kosten (Lebensmittel, Freizeit, Transport) | Sparziele (Notgroschen, Urlaub, Anschaffungen) | Automatische Monatsbilanz (Einnahmen - Ausgaben)
