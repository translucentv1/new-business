# Spec: Jane Eyre (1260) als verkaufbares Produkt

**Problem (Kaeufer):** Suchende nach "Jane Eyre eBook / kostenlos lesen /
Public Domain" finden kein Produkt dieses Funnels.
**Loesung:** Buch 1260 aufbereiten (Gliederung + Register), Preview +
Download-Deliverable + Stripe-Link, in die Live-Seite aufnehmen.
**User Stories:**
- Als Leser finde ich eine SEO-Landingpage mit Leseprobe und Kaufbutton.
- Als Kaeufer werde ich nach Zahlung zum Download (gehashter Pfad) geleitet.
**Implementation/Testing:**
- `pd_processor.build_product('1260')`, danach INHALT pruefen (Kapitelzahl
  plausibel, Titel = Inhalt, keine PG-Referenz, keine [Illustration] im TOC).
- `landingpage_gen.build()` (Preview + Deliverable), `is_corrupt` muss False.
- `stripe_uploader.create_link('1260')` → URL in `stripe_links.json`.
- `test_download_gate.py` + `test_pd_processor.py` MUSS gruen bleiben.
- `publish_site.py` → gh-pages push; Live-Check `curl` HTTP 200 auf
  `/1260/` und `/dl/<hash>/jane-eyre.html`.
**Out of Scope:** 25913-Reparatur (USER), EPUB.
